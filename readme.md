## UserDAO
Simple console program for users managment in local database. This project is based on DAO Pattern.

![userDAO](https://user-images.githubusercontent.com/97983923/169334850-85a63db7-847a-4783-a0cf-d1c0bc69af9c.jpg)

## Program usages
This program is used for users management in local database by functions like:
* add new user
* upload user data
* delete user
* list all users 

## Technologies
* Java
* SQL
